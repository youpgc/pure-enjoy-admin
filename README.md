# Pure Enjoy Admin

纯享管理后台 - 基于 React + TypeScript + Ant Design 的管理后台系统。

## 技术栈

- React 18
- TypeScript
- Vite
- Ant Design 5
- React Router 6
- Recharts (数据可视化)

## 功能模块

- 数据概览 Dashboard
- 用户管理
- 消费记录
- 心情日记
- 体重记录
- 笔记本
- 小说书架
- 版本管理

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
